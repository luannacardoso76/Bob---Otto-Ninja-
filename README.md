# Bob---Otto-Ninja-
Projeto acadêmico do Robô Otto Ninja Humanoide “Bob”, desenvolvido na disciplina Informática Aplicada à Engenharia. O trabalho envolve modelagem com Redes de Petri, impressão 3D, montagem mecânica, integração eletrônica com Arduino Nano e programação embarcada para controle de movimentos e sensores.
